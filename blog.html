1. Difference between map(), filter(), find(), forEach()

Answer:

map(): map() is a javascript function that transform the array of an elements using callback function for each element and return a new array.

filter(): filter() is a built-in function of javascript that return a new array on a condition by a callback function

forEach(): forEach() is a function that provide function once for each element and doesn't modify original array, but it doesn't return an array

find(): find() find() also a built in function of javascript that satisfy a specific condition and returns only the first matching element and it also doesn't return an array.


2. Difference between local storage, session storage and cookies

Answer:

LocalStorage: LocalStorage is a built-in web browser system which allows to store Key and Value pairs on the user's specific browser.LocalStorage data never expires.

Session Storage: SessionStorage is similar to LocalStorage allows to store Key and Value pairs on the user's browser, but only accessible for the duration of the user's use the browser.It is only accessible during the browsing session.Once the user close the browser the SessionStorage data is deleted.

Cookies: Cookies are small text files that a website stores on a user's device. They are commonly used to track user activity and preferences on a website. The website can access the cookie and read it when the user revisits the website in the future.
